# はじめに

``Android Studio``は、Androidアプリを開発するための統合開発環境<sub>(IDE)</sub>です。アプリ開発の他に、``Android SDK``の管理や``Android Emulator``といった機能も備えており、``Android Studio``一つで、複数のAndroid機器に対応したアプリを開発できます。
